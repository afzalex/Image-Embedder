About

The purpose of the project is to provide some common information like weather forecasting, today’s day-date etc to the user as easily as possible, without any using any gadget or browsing on internet. Image Embedder can alter user’s wallpaper and add some very basic information into the image. It can add any image from a location that user will provide, which may be changing in random time. User can add current weather information of his area and weather forecasting information into his desktop’s wallpaper. We can also add reminders, events, calendar’s information into the image. What if a user want to get today’s date, next day’s weather information, or to get reminded about some tasks or event. Then he will have to browse on internet, or use some gadget. But on the other hand, what it would be possible for him to get these informationwithout even making a click on system. Image Embedder can fulfill all these requirements without having user to do anything. Image Embedder can alter user’s wallpaper and add some very basic information into the image.
